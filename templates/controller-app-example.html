Project Details
Web Development Features Highlighted: CSS, Bootstrap, Jinja2, Regular Expressions, Responsive Design
Synopsis: This UI was created for a specific case in an environment where multiple video devices needed to be controlled, and the user needed to switch control to a different device on demand.
Problem:Confusion finding which device was being controlled at a given time and having to cross-reference the device that is located remotely.
Implementation: At the time, I wasn't proficient in Javascript, so I coded around different problems probably in the least efficient way with Jinja and CSS. Other than that, each button here was meant to be wired to a python script in the backend that would translate the information, triage and send to the proper devices. (The buttons are not wired to the python scripts in this example page)
Solution: HTML user interface that puts everything in one place for the user, and visually displays what device is being controlled at any time. This saves a lot of time, because the user does not have to keep cross-referencing the actual physical device that is stored in a remote location.
