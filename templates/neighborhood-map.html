Project Details
Web Development Features Highlighted: CSS, Bootstrap, KnockoutJS, Responsive Design, Google Maps API, JSON, Javascript, Jquery
Synopsis: Exercise from the Udacity program that implemented the combination of the Google Maps API, JSON data, jquery and Javascript
Problem:Create a custom that takes location data from a JSON file and dynamically displays their information in both a list and a map. List needs to be searchable by title.
Implementation: JSON data was prepopulated with location titles and their respective lat/lng coordinates. That data was then passed through the app. In the app, knockoutJS was used to dynamically show each location item and also trigger animations on the map. Lastly, the Foursquare API was introduced to add more data for the user.
Solution: Google map with a box that dynamically displays location data that is searchable.
